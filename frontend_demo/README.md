# Frontend Demonstration of the Multi-Agent System

This is a very simple website that depicts how the multi-agent system works in the background. It is mainly used as a visualization to demonstrate what the current backend system is capable of doing and not by any means an iteration in the design process of the thesis.

It connects to the `agentServer` WebSocket and displays its output in a very simple ChatGPT format.

**Note:** This frontend code has been written with extensive assistance of a Language Model
